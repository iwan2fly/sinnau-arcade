# 🕹️ Sinnau Arcade (신나유 오락실) 프로젝트 기획안

기존 프로젝트의 알짜배기 아이디어를 바탕으로, 사용자님의 직무 역량 강화까지 고려한 **React + Spring Boot** 기반의 신규 캐주얼 게임 & 커뮤니티 프로젝트를 구축합니다!

---

## 🎯 기술 스택 및 아키텍처 (확정 ✅)

> [!IMPORTANT]
> **1. 백엔드 (API 서버): Spring Boot (Java)**
> - 기존 `platform` 프로젝트에서 구현해둔 훌륭한 로직(이메일 인증, JWT 로그인, 코인 시스템 등)을 그대로 가져와서 API 전용 서버로 탈바꿈합니다. (불필요한 Thymeleaf 관련 코드는 모두 제거하여 가볍게 만듭니다.)
>
> **2. 프론트엔드 (웹 UI): React (Vite)**
> - 현업에서 널리 쓰이는 React를 채택하여 실무 감각을 익힙니다.
> - **디자인 방향성:** 플랫폼 전체는 정갈하고 모던한 UI를 유지하되, 인게임 화면에서는 애니메이션과 인터랙션을 적극 활용하여 '진짜 게임하는 맛'을 끌어올립니다.
